linkSpecials

  1. Markdown has features like bold, italic, strikethrough, bullet points and the one I am using right now, numbered points.
  2. Markdown is used in many source code hosters like GitHub, GitLab, and maybe more? It is also used in Jekyll as well as CODEDOC for blogs.
  3. Markdown has many flavours. The most comonly used Markdown flavor is GitHub Flavored Markdown. The whole lesson in Markdown will be based on GitHub Flavored Markdown that is used in a lot of things in GitHub as well as CODEDOC markdown found in the raw file edition. issues, PRs, ./README.md files and more in GitHub

To get things started, I have put a section set with the most important snippets for your Markdown perfection.

linkLearn section

linkBold and italic

Bold can be used to bolden a text while italic is... well, you know.

bold+italic.md
1link**This is bold**

2link*This is italic*

3link__This is also bold__

4link_And this is also italic_

5linkWhen combined, you get ***this*** or __*this*__ or even **_this_**

The result:

This is bold

This is italic

This is also bold

And this is also italic

linkWhen combined, you get this or this or even this

linkLinks

Links are a fun way to share links in suspense way and in different ways too. You can highlight it like code highlights too. Bold italic are main here. Let's go(claps like GothamChess)!

links.md
1link[homepage redirect](https://coderlg.coding.blog) <!--> Basic syntax of links -->

2link[`homepage redirect with code highlight`](https://coderlg.coding.blog) <!-->Code highlighted-->

3link[***hi***](https://coderlg.coding.blog) <!-->bold and italic, either one or both in any combination :)-->

The result: homepage redirect

homepage redirect with code highlight

hi

linkTask list

Tasks are a way in Markdown, especially the GitHub flavored one to make a checklist. It can be useful in a task list for a new feature in GitHub.

tasks.md
1link- [ ] put a space in the parenthesis

2link- [x] tasks that are done have an x instead of a space inside the parenthesis

3linkthis is in code mode. let's switch now!

Result(put this in Typora):

  • put a space in the parenthesis
  • tasks that are done have an x instead of a space inside the parenthesis

linkHeaders

Headers are useful in Readmes for bold text with just this: # used as:

Headers.md
1link# h1

2link## h2

3link### h3

4link#### h4

5link##### h5

6link###### h6

Result:

linkh1

linkh2

linkh3

linkh4

linkh5
linkh6

only 6 sizes of headers

linkStrikethrough

This is the only piece which needs the HTML tag <s>whatever</s>

strikethrough.md
1link<s>test</s>

Result: test

linkBulleted and numbered points

linkBulleted points

Bulleted points can be used if you want a concise list of points if you are studying.

bulleted-points.md
1link- hi

2link- my name is LG

3link- :laugh: :+1:

Result:

  • hi
  • my name is LG
  • :laugh: :+1:

linkNumbered points

Same way you wrote questions and answers when you were young(not a joke)

numbered-points.md
1link1. hi

2link2. bye

3link3. joking

Result:

  1. hi
  2. bye
  3. joking

linkImages

Images are interesting. Basically, you add it like a link but do it like this:
